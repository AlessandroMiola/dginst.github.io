OPENTIMESTAMPS: STEP BY STEP TUTORIAL

GitHub project: https://github.com/fametrano/javascript-opentimestamps-tools

1. HASH: get hash value from data file (no internet connectivity required)



2. STAMP: get hexadecimal OTS from hash (internet connectivity required)


3. SAVE: save hexadecimal OTS as OTS file (no internet connectivity required)


4. LOAD: load hexadecimal OTS from OTS file (no internet connectivity required)


5. INFO: display info from hexadecimal OTS (no internet connectivity required)


6. UPGRADE: upgrade hexadecimal OTS (internet connectivity required)


7. VERIFY: get attestations from hexadecimal OTS (internet connectivity required)
